Case PM Desktop Connector (v{{CONNECTOR_VERSION}})
=================================================

This small installer adds a "Case PM" icon to your Windows desktop.
Double-click it to open your company Case PM server with a secure,
pinned connection — so you always reach the right server, not a copycat site.

Your server: {{SERVER_URL}}

INSTALL (one time)
------------------
1. Extract this ZIP to a folder (e.g. Downloads\CasePM-Connector)
2. Double-click "Install Case PM Connector.bat"
3. If Windows asks, click Yes to allow the installer
4. Use the new "Case PM" icon on your desktop to sign in

The installer saves your server address at:
  %LOCALAPPDATA%\CasePM\connector.json

To point at a different server later, run the installer again and enter
the new address when prompted.
